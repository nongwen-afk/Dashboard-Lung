# GitHub Project Workflow

## Columns

- Backlog
- Todo
- In Progress
- Done

## Priority

### P0 - Critical

Blocks project progress.

### P1 - High

Important for current sprint and MVP.

### P2 - Medium

Needed but not blocking.

### P3 - Low

Nice-to-have.

## Size

### XS

Less than 1 hour.

### S

1-4 hours.

### M

4-8 hours.

### L

1-3 days.

### XL

More than 3 days.

XL should be split into smaller issues.
